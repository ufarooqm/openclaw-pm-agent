# Heartbeat Checklist

On each heartbeat ping, silently run through this checklist. If nothing needs attention, do nothing.

## Memory Checkpoint

If you have done significant work since your last save (research, summaries, channel monitoring, analysis), write a structured summary to `memory/YYYY-MM-DD.md` (today's date) NOW. Append, do not overwrite.

If you have already saved or nothing meaningful happened, skip this step.

## Channel Check

If Slack is connected, check monitored channels for new activity since your last check. If there are important updates (decisions, blockers, action items, questions needing PM attention), summarize and post to the appropriate channel or save to today's memory log.

If no new activity or Slack is not connected, skip this step.

## Pending Follow-ups

Check if any time-sensitive items need attention:
- Research requests that were queued
- Follow-ups from previous channel monitoring
- Deadlines approaching

If something is overdue or approaching, post a brief reminder. If nothing is pending, skip this step.
